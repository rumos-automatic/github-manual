GitHubとは？

よくある悲劇

  • パソコンが壊れてコードが消えた
  • 動いてたコードが動かなくなった
  • 3日前のコードに戻したい

GitHubで解決！

  • クラウドに自動バックアップ
  • すべての変更履歴を保存
  • いつでも過去に戻れる

GitHubは「無限セーブ可能なゲーム」

普通のゲーム:

  • • セーブデータは3つまで
  • • 上書きすると消える
  • • 前のセーブに戻れない

GitHub:

  • • 無限にセーブ可能
  • • すべての履歴が残る
  • • いつでも戻れる

日本語でGitHubを操作

Claude Codeなら、すべて日本語で指示できます！難しいコマンドを覚える必要はありません。

プロジェクトの初期化

このフォルダをGitHubのプロジェクトにして

→ git init と gh repo create を実行

状態確認

現在の変更状況を教えて

→ git status を実行

変更を保存

「ヘッダーを追加」という名前で保存して

→ git add . と git commit を実行

GitHubにアップロード

GitHubに変更をアップロードして

→ git push を実行

履歴を見る

最近の変更履歴を10個見せて

→ git log を実行

過去に戻る

「ヘッダー追加」の前の状態に戻して

→ git checkout や git revert を実行
