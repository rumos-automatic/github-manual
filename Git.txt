# 🚀 日本語だけでGitHub完全マスター！Claude Code活用ガイド

## 📖 GitHubとは？

**GitHub（ギットハブ）**は、プログラムのコードをインターネット上に保存・共有できるサービスです。

### なぜGitHubが必要なの？

プログラミングでよくある悲劇：
- 😱 「パソコンが壊れてコードが全部消えた...」
- 😭 「動いてたコードを変更したら動かなくなった...」
- 🤔 「3日前のコードに戻したい...」

**GitHubがあれば全部解決！**

---

## 🎮 GitHubは「無限セーブ可能なゲーム」

### ゲームのセーブとGitHubの違い

**普通のゲーム：**
- セーブデータは3つまで
- 上書きすると前のデータは消える
- 前のセーブに戻りたくても無理...

**GitHub：**
- **無限にセーブポイントを作れる**
- **すべての履歴が残る**
- **いつでも好きな時点に戻れる**
- **クラウド保存で安全**

## 🚀 準備編：GitHubを使い始める

### ステップ1：GitHubアカウントを作る（5分）

1. [GitHub.com](https://github.com)にアクセス
2. 「Sign up」をクリック
3. 以下を入力：
   - **Username**: あなたのユーザー名（例：tanaka-taro）
   - **Email**: メールアドレス
   - **Password**: パスワード（8文字以上）

### ステップ2：GitHub CLIをインストール（Claude Codeにお任せ！）

Claude Codeを起動：
```bash
claude
```

そして日本語で指示するだけ：
```
GitHub CLIをインストールして。
私のパソコンは[Windows/Mac]です。
```

**Claude Codeが自動的に：**
- ✅ OSに合った方法を選択
- ✅ インストールコマンドを実行
  - Mac: `brew install gh`
  - Windows: `winget install --id GitHub.cli`
  - Ubuntu: `sudo apt install gh`
- ✅ インストール成功を確認

### ステップ3：GitHub認証設定（超重要！）

インストールが終わったら、続けて指示：
```
GitHub CLIの認証設定をして。
ブラウザで認証する方法でお願い。
```

**Claude Codeが実行：**
```bash
gh auth login
```

**認証プロセス：**
1. Claude Codeが対話形式で選択肢を選んでくれる
   - GitHub.com を選択
   - HTTPS を選択
   - Browser認証を選択
2. **ブラウザが自動的に開く**
3. **あなたがやること：**
   - GitHubにログイン（まだの場合）
   - 「Authorize GitHub CLI」をクリック
   - 緑色の認証コードが表示される
4. ターミナルに戻ると自動的に認証完了！

**確認：**
```
GitHubの認証ができてるか確認して。
私のユーザー名も表示して。
```

Claude Codeが実行：
```bash
gh auth status
```

成功していれば、あなたのGitHubユーザー名が表示されます！
